# Countdown Timer: Going Home

A simple, accessible, and responsive countdown timer targeting **Monday, April 27, 2026**. Designed to be hosted on GitHub Pages with zero dependencies.

## Features
- **Modern UI:** Clean, mobile-first design.
- **Accessible:** Built with semantic HTML and ARIA standards (WCAG 2.2 AA compliant).
- **Lightweight:** Pure HTML, CSS, and Vanilla JavaScript.
- **GitHub Pages Ready:** Optimized for static hosting.

## Development Standards
This project follows best practices from `github/awesome-copilot`, including:
- Semantic HTML structure.
- CSS Custom Properties for maintainability.
- Modular, defensive JavaScript.
- Accessibility-first development.

## Project Structure
- `index.html`: Main entry point and semantic structure.
- `style.css`: Responsive, high-contrast styling.
- `script.js`: Countdown logic and DOM updates.
- `spec/`: Project documentation and roadmap.
- `.github/copilot-instructions.md`: Project-specific AI guidelines.

## Deployment
1. Push this repository to GitHub.
2. Go to repository **Settings** -> **Pages**.
3. Select the `main` branch as the source.
4. Your countdown timer will be live at `https://<username>.github.io/<repo-name>/`.
